# Food Delivery System – Web Project

## Project Description  
The Food Delivery System is a simple web-based application developed using HTML, CSS, and JavaScript. It allows users to browse food items, add them to the cart, and place orders. This project is created for learning and practice purposes to strengthen frontend development skills.

---

## Technologies Used  
- HTML – Page structure  
- CSS – Styling and layout  
- JavaScript – Interactivity and logic  

---

## Features  
- User-friendly interface  
- Food menu display  
- Add to cart functionality  
- Order summary  
- Simple and clean design  
- Responsive layout (basic)  

---

## Learning Outcomes  
- Improved understanding of web development basics  
- Hands-on experience with DOM manipulation  
- Better logic-building skills  
- Practical project implementation
